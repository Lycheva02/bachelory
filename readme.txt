Для воспроизведения работы необходимо установить jupyter (или воспользоваться другой средой для работы с jupyter-ноутбуками), python3.10.11, а также библиотеки:
pandas: 2.0.1
scikit-learn (sklearn): 1.0.2
numpy: 1.22.4
joblib: 1.1.0
matplotlib: 3.5.3
seaborn: 0.12.0

Для запуска jupyter в директории с ноутбуком выполняется команда:
jupyter notebook
Далее необходимо выбрать требуемый ipynb-файл для запуска через интерфейс Jupyter. Последующая работа выполняется через интерактивный интерфейс.

Для запуска интерактивного дашборда дополнительно требуется библиотека:
streamlit: 1.33.0
Далее из директории, в которой расположен файл dashboard.py выполняется команда:
streamlit run dashboard.py
Дальнейшее взаимодействие осуществляется через интерактивный интерфейс.
